更方便、更快捷、更深入帮助您分析Github的开源项目

GitPlugin, 一款简易方便的google浏览器插件，在每一个github的开源仓库页面，我们帮您分析该仓库的各种参数，判断该仓库在github所有开源项目中所处的地位，并伴有各类分析图表，简明清晰地展示分析数据，让您更快速地了解该仓库，并为您推荐符合喜好的仓库。我们致力于实现开源项目的最高效推广，致力于让每一位代码爱好者找到适合自己的仓库

watch数、star数、fork数全局比较, subscribe跟踪项目进行

对于每一个用户打开的仓库，计算其在整体数据中的地位， 让用户对于该仓库所处水平有一个整体感知。提供subscribe选择， 用户订阅后，利用cookies存储，跟踪项目进展动态

通知、仓库新闻、码友评论、图表分析

插件弹出框的四大组成部分，包括用户订阅仓库后，仓库更新的通知、有关该仓库 在各个开源社区的版本发布或者讨论新闻、该仓库收到的评论、对该仓库的数据分 析图表。

新闻与评价

展示项目相关的新闻与评价，包括新闻关键词， 并利用bosonnlp提供的API进行情感分析，计算新闻和评论的积极因子

对于该仓库使用的语言进行统计，不同的仓库可能使用多种语言开发， 确保用户了解所浏览项目开发语言的构成对于用户选择项目很有必要。

Popularity,Contributor分析图
